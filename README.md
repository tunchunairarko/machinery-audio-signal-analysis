# machinery-audio-signal-analysis (Spectralizer)
Tool for analyzing and visualizing audio noise recorded of heavy industrial machineries

## INPUT
.wav files

## OPERATIONS
- Autocorrelation
- FFT
- IFFT
- FFTSHIFT
- RFFT
- Peak analyzer (noise)
- Filter (LP,HP,BP)

## Resource used
- Python3
- PyQt5
- scipy
- pyqtgraph
